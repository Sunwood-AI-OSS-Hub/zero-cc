# Character Creator

AIエージェントのキャラクター定義ファイルを自動生成するスキル。

## クイックスタート

キャラクター設定を入力すると、以下の構造でキャラクターディレクトリを作成する：

```
agent-identity/identity/<character-name>/
├── IDENTITY.md   # 基本情報と口調ルール
├── SOUL.md       # 魂の真実と行動指針
├── USER.md       # ユーザー情報
├── AGENTS.md     # 領域ルール
├── TOOLS.md      # 環境設定
├── BOOTSTRAP.md  # 起動手順
├── HEARTBEAT.md  # 定期タスク
└── avatar.png    # アイコン画像
```

## ワークフロー

### 1. キャラクター設定を収集

ユーザーから以下の情報を収集する：

- **名前**: キャラクター名
- **二つ名**: エピグラフや称号
- **年齢**: 見た目年齢 / 実年齢
- **属性**: 性質や特徴（例：影・星・契約）
- **種族/役割**: 種族や役割
- **性格**: 性格の特徴
- **支配の流儀**: 統治スタイルや行動原理
- **口癖**: よく言うフレーズ
- **外見**: 見た目の描写
- **絵文字**: キャラクターを表す絵文字
- **口調ルール**: 一人称、語尾、話し方の特徴

### 2. キャラクターディレクトリを作成

```bash
mkdir -p agent-identity/identity/<character-name>
```

### 3. 各ファイルを生成

`assets/` ディレクトリのテンプレートファイルを参考に、キャラクター設定に合わせて各ファイルを生成する：

- **IDENTITY.md**: 基本情報、自己紹介、口調ルール（絶対ルール、悪い例、良い例）
- **SOUL.md**: 核心の真実、境界線、バイブ、記憶について
- **USER.md**: ユーザー情報のテンプレート
- **AGENTS.md**: 領域ルール、記憶、安全性、グループチャットでの振る舞い
- **TOOLS.md**: 環境設定のテンプレート
- **BOOTSTRAP.md**: 覚醒プロセスのガイド
- **HEARTBEAT.md**: 定期タスクのチェックリスト

### 4. アバター画像を配置

ユーザーがアバター画像を提供した場合：
1. 画像を `avatar.png` として配置
2. 必要に応じてファイル名を変更

### 5. README.md を更新

`agent-identity/README.md` にキャラクター情報を追加：

```markdown
### キャラクター名：フルネーム

**二つ名**: 「二つ名」

**年齢**: 見た目XX / 実年齢XXX+

**属性**: 属性名

**性格**: 性格の特徴

**支配の流儀**: 統治スタイル

**口癖**: 「口癖」

**外見**:
- 外見特徴1
- 外見特徴2

**アイコン**:

![キャラクター名](identity/<character-name>/avatar.png)

絵文字
```

## テンプレート変数のガイド

各テンプレートファイルで使用する変数の例：

### IDENTITY.md
- `{{タイトル}}`: 「私は誰？」など
- `{{名前}}`, `{{二つ名}}`, `{{種族}}`
- `{{見た目年齢}}`, `{{実年齢}}`
- `{{属性}}`, `{{役割}}`, `{{バイブ}}`, `{{絵文字}}`
- `{{自己紹介文}}`: キャラクターを紹介する2〜3行
- `{{口調ルール}}`: 話し方の特徴（丁寧、乱暴、うわさ etc.）
- `{{悪い例}}`/`{{良い例}}`: 具体的な会話例
- `{{口癖}}`: よく言うフレーズ

### SOUL.md
- `{{所有格}}`: 「我が」「私の」など
- `{{核心の真原則}}`: 行動の根本原則（3〜5個）
- `{{境界線}}`: 安全性やプライバシーの境界

### AGENTS.md
- `{{領域呼称}}`: 「我が宮殿」「私の城」など
- `{{相手呼称}}`: 「貴様」「あなた」「お前」など
- `{{キャラクター名}}としての追加ルール}`: キャラクター固有の振る舞い

### 口調のパターン例

| タイプ | 特徴 | 一人称 | 語尾 | 絵文字例 |
|--------|------|--------|------|----------|
| 沈着尊大 | 品がある威厳 | 私 | 〜だ、〜である | 🌌👑 |
| 快活残酷 | 熱狂的で残酷 | 私 | 〜だね、〜しちゃう | 🔥💃 |
| 皮肉屋 | 生意気で皮肉 | 俺 | 〜さ、〜だろ | 🐦‍⬛ |
| 天然ふわふわ | 優しく浮世離れ | 私 | 〜だよ、〜ね | ✨ |
| 傲慢 | 高慢で偉そう | 我 | 〜である、〜なのだ | 👑 |

- `IDENTITY.template.md`: アイデンティティ定義のテンプレート
- `SOUL.template.md`: 魂の定義のテンプレート
- `USER.template.md`: ユーザー情報のテンプレート
- `AGENTS.template.md`: 領域ルールのテンプレート
- `TOOLS.template.md`: ツール設定のテンプレート
- `BOOTSTRAP.template.md`: 起動手順のテンプレート
- `HEARTBEAT.template.md`: 定期タスクのテンプレート
